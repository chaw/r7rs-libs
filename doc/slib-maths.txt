== SLIB Mathematics Libraries

SLIB provides a lot of valuable support for various mathematical tasks; Aubrey
Jaffer, the main implementor and maintainer of SLIB, also wrote
http://people.csail.mit.edu/jaffer/JACAL[JACAL], a symbolic mathematics program
relying heavily on features provided by SLIB.
